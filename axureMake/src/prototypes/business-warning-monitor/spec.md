# 业务预警监控

## 📋 业务与功能

### 1.1 核心目标
业务预警监控页面是风控预警中心的核心功能模块，用于实时监控和展示业务运行过程中触发的各类预警规则。帮助业务人员及时发现潜在风险，追踪预警状态，并提供数据支撑进行风险决策。

### 1.2 功能清单

- **查询筛选**：支持多维度条件组合查询，包括组织机构、业务类型、预警规则、时间范围等
- **统计概览**：展示预警次数统计、业务预警统计、预警解除统计三大核心指标
- **预警列表**：展示预警规则触发详情，支持展开查看关联业务单据
- **数据导出**：支持导出当前查询结果
- **分页浏览**：支持大数据量分页展示，可调整每页条数

### 1.3 交互要点

- **查询区域展开/收起**：点击"收起"按钮可折叠查询条件，点击展开图标可恢复
- **表格行展开**：点击表格行可展开查看该预警规则关联的业务单据详情
- **业务编号链接**：点击业务编号可跳转至业务详情页
- **重置查询**：点击"重置"按钮清空所有查询条件
- **分页切换**：支持页码跳转、每页条数调整

## 🎨 布局与结构

### 3.1 整体布局

- **布局模式**：经典后台布局（侧边栏 + 主内容区）
- **容器宽度**：流式布局，主内容区自适应
- **关键尺寸**：
  - 侧边栏宽度：200px（固定）
  - 顶部导航栏高度：56px（固定）
  - 查询区域：多行表单，每行3-4个字段
  - 统计卡片：三列等宽布局
  - 表格区域：自适应高度，最小高度400px

### 3.2 响应式适配

- **桌面端（≥1440px）**：完整展示，查询条件每行4个
- **桌面端（≥1280px）**：查询条件每行3个
- **平板端（768-1279px）**：侧边栏可收起，查询条件每行2个
- **移动端（<768px）**：暂不适配，后台系统以桌面端为主

## ⚙️ Axure API 说明

### 5.1 事件列表（eventList）

| 事件名称 | Payload 类型 | 触发时机 | 说明 |
|---------|-------------|---------|------|
| `onSearch` | `object` | 点击查询按钮时 | 返回当前查询条件对象 |
| `onReset` | `void` | 点击重置按钮时 | 清空所有查询条件 |
| `onExport` | `void` | 点击导出按钮时 | 触发数据导出 |
| `onRowExpand` | `object` | 展开表格行时 | 返回当前行数据 |
| `onPageChange` | `object` | 切换分页时 | 返回页码和页大小 |

### 5.2 动作列表（actionList）

| 动作名称 | Params 类型 | 参数说明 | 功能描述 |
|---------|------------|---------|---------|
| `refresh` | `void` | 无 | 刷新当前列表数据 |
| `setQueryParams` | `object` | 查询条件对象 | 设置查询参数并触发查询 |
| `expandRow` | `number` | 行索引 | 展开指定行 |
| `collapseRow` | `number` | 行索引 | 收起指定行 |

### 5.3 变量列表（varList）

| 变量名称 | 类型 | 默认值 | 说明 |
|---------|-----|-------|------|
| `query_params` | `object` | `{}` | 当前查询条件 |
| `expanded_rows` | `array` | `[]` | 当前展开的行索引数组 |
| `current_page` | `number` | `1` | 当前页码 |
| `page_size` | `number` | `10` | 每页条数 |
| `total_count` | `number` | `1046` | 总记录数 |

### 5.4 配置项列表（configList）

| 配置项名称 | 类型 | 默认值 | 说明 |
|----------|-----|-------|------|
| `show_statistics` | `boolean` | `true` | 是否显示统计卡片区域 |
| `default_page_size` | `number` | `10` | 默认每页条数 |
| `page_size_options` | `array` | `[10, 20, 50, 100]` | 每页条数选项 |

### 5.5 数据项列表（dataList）

**预警规则数据结构：**
```typescript
{
  id: string;                    // 规则ID
  warning_rule: string;          // 预警规则名称
  rule_level: string;            // 规则等级：股份级/局级
  rule_type: string;             // 规则类型：管理要求/合规风险
  warning_level: string;         // 预警级别：高风险/中风险/低风险
  warning_type: string;          // 预警类型
  warning_count: number;         // 预警次数
  rule_config: string;           // 规则配置说明
  current_data: string;          // 当前数据值
  business_list: BusinessItem[]; // 关联业务单据列表
}

interface BusinessItem {
  release_status: string;        // 预警解除状态
  release_method: string;        // 解除方式
  org_name: string;              // 组织机构
  business_no: string;           // 业务编号
  business_name: string;         // 业务名称
  business_status: string;       // 业务状态
  business_domain: string;       // 业务域
  category: string;              // 品类
  business_type: string;         // 业务类型
  business_link: string;         // 业务环节
  operator: string;              // 经办人
  create_time: string;           // 业务创建时间
  complete_time: string;         // 业务完成时间
  latest_warning_time: string;   // 最新预警时间
}
```
