# 招标列表页面规格文档

## 1. 页面基本信息

- **页面名称**: 招标列表
- **页面路径**: `src/prototypes/bidding-list/index.tsx`
- **业务模块**: 招标采购
- **视觉风格**: 蓝色系企业级管理后台 (参考集采工作台风格)

## 2. 页面布局架构

### 2.1 整体布局
- **侧边栏 (Sider)**: 固定宽度 80px (极简模式图标导航)，背景深蓝色。
- **头部 (Header)**: 高度 64px，背景白色，包含 Logo、面包屑、搜索框、用户信息。
- **主内容区 (Content)**: 背景色 #f0f2f5，内边距 16px。

### 2.2 模块划分
- **面包屑导航**: 位于头部下方，显示“招标采购 > 招标列表”。
- **查询过滤区**: 卡片样式，包含多行表单，支持展开/收起。
- **操作工具栏**: 位于查询区下方，包含动作按钮（开启招标等）和统计信息。
- **数据表格区**: 核心展示区域，包含分页。

## 3. 组件详细定义

### 3.1 顶部导航栏 (Header)
- **左侧**: 集采工作台 Logo。
- **中间**: 搜索框 (支持通过关键字搜索菜单)。
- **右侧**: 简体中文切换、云筑首页、寻源工作台、发布招募需求、切换新版、收藏、下载、帮助、消息(带红点71)、用户头像及姓名。

### 3.2 侧边栏菜单 (Sider)
- **图标 + 文字**: 
  - 首页、权限、采购、分供、招标 (选中状态)、工程、联采、合同、投诉、清欠、收验。

### 3.3 查询表单 (Query Form)
- **字段 1**: 组织机构 (Select) + 包含下级 (Checkbox)。
- **字段 2**: 招标编号 (Input) + 精确搜索 (Checkbox)。
- **字段 3**: 招标名称 (Input)。
- **字段 4**: 招标状态 (Select) + 包含已废标 (Checkbox)。
- **字段 5**: 招标方式 (Select)。
- **字段 6**: 合作品类 (Select)。
- **字段 7**: 经办人 (Input) + 本人经办 (Checkbox)。
- **操作**: 查询 (蓝色)、重置 (边框)、展开 (链接)。

### 3.4 操作栏 (Action Bar)
- **按钮组**: 
  - 开启招标 (Primary)
  - 从废旧物资申请发起招标 (Default)
  - 从投标方推荐发起招标 (Default)
  - 导出 (Default)
  - 设置 (Icon)
- **信息展示**: 本页 10 条，总计 1408 条。

### 3.5 数据表格 (Data Table)
- **列定义**:
  - 状态: 文本 (如: 招标文件中编制中、投标中、招标完成)
  - 招标编号: 蓝色链接 + 图标
  - 招标名称: 文本
  - 组织机构: 文本
  - 使用项目: 文本
  - 区域: 文本
  - 招标方式: 文本
  - 招标品类: 文本
  - 经办人: 文本
  - 操作: 查看、更多 (蓝色链接)

## 4. 视觉规范
- **主色**: #1890ff
- **字体**: 14px (主要), 12px (辅助)
- **表格行高**: 54px (预计)
- **边框颜色**: #f0f0f0
