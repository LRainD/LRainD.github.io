# 招标列表页面规格说明 (Spec)

## 业务与功能
- **定位**：企业级集采工作台的招标列表页面。
- **核心目标**：提供招标项目的查询、筛选、列表展示及操作入口。
- **功能清单**：
  1. 全局导航：包含侧边栏菜单和顶部导航条。
  2. 高级搜索：多条件组合查询，包括输入框、下拉框、复选框。
  3. 操作按钮组：发起各类招标、导出数据、表格设置。
  4. 数据表格：展示招标项目明细，包含状态、编号、名称、机构、项目、区域、方式等，支持查看和更多操作。

## 内容规划
- **信息架构**：
  - 左侧边栏 (Sidebar)：一级功能导航模块。
  - 顶部导航 (Header)：系统Logo、全局搜索、语言切换、快捷入口、用户中心。
  - 主内容区 (Main Content)：
    - 面包屑导航
    - 高级查询面板
    - 表格工具栏及分页信息
    - 数据列表

## 布局与结构
- **整体布局**：经典的左中右/上下分栏结构。左侧固定侧边栏(宽约80px)，右侧为主操作区；主操作区分为顶部固定Header(高约56px)和下方滚动内容区。
- **模块划分**：
  - 查询区采用网格布局，每行3-4个表单项，右下角为操作按钮。
  - 列表区为自适应宽度的表格。

## 视觉规范
- **主题风格**：典型的 ToB 管理系统风格。
- **色彩规范**：
  - 侧边栏背景：深蓝色（近似 `#1858a8`）
  - 主色调（按钮/链接/高亮）：蓝色（近似 `#1677ff` 或 `#0066cc`）
  - 背景色：主内容区背景浅灰（近似 `#f0f2f5` 或 `#f5f7fa`），模块背景纯白。
  - 文本颜色：标题与主要文本深色，次要文本灰黑色。
- **字体与图标**：使用无衬线字体，搭配 Lucide 图标库。
