# 企业关联检测列表

## 表格字段

| 字段 | 说明 |
|------|------|
| 序号 | 从1开始的顺序编号 |
| 企业关联检测编号 | 蓝色可点击链接，格式如 AAD2026041400001669970 |
| 关联关系 | 显示"存在企业关联关系的企业：X家"，数字标蓝 |
| 状态 | 检测完成（绿色 #52C41A）/ 检测超时（红色 #FF4D4F） |
| 检测发起时间 | 格式如 2026-04-14 20:47:37 |
| 权益使用 | 显示"消耗企业关联检测额度：X家"，数字标蓝 |
| 操作 | 查看结果（蓝色链接），超时记录额外显示"重新检测" |

## 操作说明

- **查看结果**：点击后跳转至检测详情页
- **重新检测**：仅对超时记录显示，点击后重新发起检测流程

## 数据示例

```typescript
{
  id: '1',
  status: 'completed', // 'completed' | 'timeout'
  detectNo: 'AAD2026041400001669970',
  relation: '存在企业关联关系的企业：2家',
  startTime: '2026-04-14 20:47:37',
  quotaUsage: '消耗企业关联检测额度：2家'
}
```
