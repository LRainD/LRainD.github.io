# 企业关联检测列表

## 📋 业务与功能

### 1.1 核心目标
企业关联检测列表页面是云筑AI工坊平台的核心功能模块之一，用于展示用户发起的所有企业关联检测结果。用户可以查看检测状态、关联关系详情、发起时间等信息，并支持查看详情和重新检测操作。

### 1.2 功能清单
- **列表展示**：以表格形式展示企业关联检测记录
- **状态筛选**：按检测状态筛选（检测完成、检测超时）
- **查看详情**：点击"查看"按钮查看检测详情
- **重新检测**：对超时或需要重新检测的记录发起重新检测
- **分页导航**：支持翻页浏览历史记录

### 1.3 交互要点
- **菜单选中**：当前页面"企业关联"菜单项高亮显示
- **查看操作**：点击蓝色"查看"链接跳转详情页
- **重新检测**：点击"重新检测"按钮触发检测流程
- **消息通知**：底部消息图标显示未读数量（红点标识）

## 📊 内容规划

### 2.1 信息架构
```
企业关联检测列表
├── 左侧导航栏
│   ├── LOGO区域
│   ├── 工具模块（主页、企业检测、风险解析、投标智检、企业关联、智能清标）
│   ├── 服务模块（应用市场、帮助中心）
│   └── 底部工具栏（消息通知、用户信息）
└── 主内容区
    ├── 表格标题栏
    ├── 数据表格（8列）
    └── 分页器
```

### 2.2 数据来源
- **数据类型**：列表数据
- **数据源**：生成示例数据
- **关键字段**：
  - `status`: 检测状态（检测完成/检测超时）
  - `detectNo`: 企业关联检测编号
  - `relation`: 关联关系描述
  - `startTime`: 检测发起时间
  - `createTime`: 创建时间
  - `quotaUsage`: 权益使用说明
  - `creator`: 发起人
  - `operation`: 操作按钮

### 2.3 内容示例
- 检测编号格式：AAD2026041400001669970
- 关联关系格式："存在企业关联关系的企业：X家"
- 权益使用格式："消耗企业关联检测额度：X家"
- 发起人格式：jcftest1（jcftest1）

## 🎨 布局与结构

### 3.1 整体布局
- **布局模式**：双栏（左侧固定导航 + 右侧主内容）
- **容器宽度**：流式布局，左侧固定200px，右侧自适应
- **关键尺寸**：
  - 左侧导航栏：宽度200px，高度100vh
  - 主内容区：margin-left 200px，padding 24px
  - 表格区域：宽度100%，最小高度500px

### 3.2 响应式适配
- **桌面端（≥1200px）**：完整双栏布局
- **平板端（768-1199px）**：左侧导航可折叠
- **移动端（<768px）**：左侧导航隐藏，通过汉堡菜单触发
