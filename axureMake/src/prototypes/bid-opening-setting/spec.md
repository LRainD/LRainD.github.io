# 开标设置页面规格说明 (Spec)

## 📋 业务与功能

### 1.1 核心目标
- **定位**：企业级集采工作台的开标设置页面，属于招标流程中的开标环节配置。
- **核心目标**：提供开标前的参数配置、开标人设置、流程导航及状态展示。
- **用户价值**：帮助招标经办人在开标前完成必要的设置，确保开标流程合规执行。

### 1.2 功能清单
- **[流程导航]**：展示招标全流程（约标→发标→开标→评标→定标），当前高亮"开标"步骤。
- **[开标倒计时]**：显示距投标截止的剩余时间（天/时/分/秒）。
- **[招标信息展示]**：展示招标编号、组织机构、招标方式、项目名称、品类、经办人等关键信息。
- **[开标设置面板]**：配置开标相关参数，含开标方式选择（密码+密钥）。
- **[开标人设置]**：管理参与开标的人员列表，展示账号、电话、所属组织、是否加密等信息。
- **[操作按钮]**：上一步、下一步导航；查看回标情况、查看开标记录快捷入口。

### 1.3 交互要点
- 流程步骤点击切换（约标、发标、开标、评标、定标）。
- 开标设置面板可展开/收起。
- 上一步/下一步按钮导航。

## 📊 内容规划

### 2.1 信息架构
```
开标设置页面
├── 顶部导航栏 (Header)
│   ├── Logo + 系统名称
│   ├── 组织机构选择
│   ├── 全局搜索
│   ├── 语言切换
│   └── 快捷入口 + 用户信息
├── 左侧边栏 (Sidebar)
│   └── 一级功能导航（首页/采购/招标等）
├── 面包屑导航
│   └── 招标采购 > 招标列表 > 开标设置
├── 主内容区
│   ├── 招标信息卡片
│   │   ├── 标题区（物资标签 + 招标名称 + 步骤按钮）
│   │   ├── 倒计时
│   │   └── 信息网格（编号/机构/方式/项目/品类/经办人）
│   ├── 流程导航（约标→发标→开标→评标→定标）
│   ├── 开标设置面板
│   │   ├── 面板标题 + 操作按钮
│   │   └── 开标方式（密码+密钥）
│   └── 开标人设置表格
│       ├── 开标人/账号/电话/所属组织/是否已加密
│       └── 数据行
```

### 2.2 数据来源
- **数据类型**：单据/配置
- **数据源**：生成示例数据
- **关键字段**：
  - `bidding_no`: 招标编号
  - `org_name`: 组织机构
  - `bidding_method`: 招标方式
  - `project_name`: 项目名称
  - `category`: 品类
  - `handler`: 经办人
  - `opening_method`: 开标方式
  - `opener_name`: 开标人
  - `opener_account`: 账号
  - `opener_phone`: 电话
  - `opener_org`: 所属组织
  - `is_encrypted`: 是否已加密

## 🎨 布局与结构

### 3.1 整体布局
- **布局模式**：经典管理后台布局，左侧固定边栏 + 右侧主内容区
- **容器宽度**：流式自适应
- **关键尺寸**：
  - 顶部导航栏：56px 高度
  - 左侧边栏：80px 宽度（图标式导航）
  - 面包屑：32px 高度
  - 主内容区：自适应，左右padding 24px

### 3.2 响应式适配
- **桌面端（≥1200px）**：完整布局，所有元素正常展示
- **平板端（768-1199px）**：保持布局，内容区适当收缩
- **移动端（<768px）**：暂不重点适配
