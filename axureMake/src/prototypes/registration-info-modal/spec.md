# 报名信息弹窗

- 文档编号：PRD-Registration-Info-Modal
- 负责人：[@liurundong]
- 版本：v1.0
- 状态：已完成

## 1. 摘要

报名信息弹窗，用于供应商在采购系统中提交报名信息。

## 2. 功能规格

### 2.1 联系方式
- 联系人（必填）
- 手机号（必填）
- 邮箱（必填）

### 2.2 报名附件
- 附件上传（必填）
- 支持格式：.jpeg、.jpg、.gif、.png、.doc、.docx、.xlsx 等
- 大小限制：500M
- PDF电子签支持：100M以内

### 2.3 操作按钮
- 取消
- 直接报名
- 发起电子签后报名
