# 报名信息弹窗

- 文档编号：PRD-Registration-Info-Modal
- 负责人：[@liurundong]
- 版本：v1.2
- 状态：已完成

## 1. 摘要

报名信息弹窗，用于供应商在采购系统中提交报名信息。

## 2. 功能规格

### 2.1 联系方式
- 联系人（必填）
- 手机号（必填）
- 邮箱（必填）

### 2.2 报名附件

#### 2.2.1 附件上传方式（必填）
- 单选字段，选项：
  - **扫描件上传**：传统附件上传方式
  - **电子签章上传**：需进行电子签章后再提交
- 未选择上传方式时，附件字段不显示
- 选择后展开附件字段

#### 2.2.2 附件上传
- 附件上传（必填）
- 支持格式：.jpeg、.jpg、.gif、.png、.doc、.docx、.xlsx 等
- 大小限制：500M
- PDF电子签支持：100M以内

### 2.3 操作按钮（联动规则）

按钮根据**附件上传方式**的选择进行联动：

| 附件上传方式 | 操作按钮 |
|-------------|---------|
| 扫描件上传 | 提交报名 |
| 电子签章上传 | 发起电子签后报名 |
| 未选择 | 不显示提交按钮 |

- 取消（始终显示）

### 2.4 电子签章流程

#### 2.4.1 选择文件弹窗
点击"发起电子签后报名"按钮后，弹出选择文件弹窗：
- 弹窗标题：选择文件（带橙色感叹号图标）
- 说明文字：请勾选您需进行电子签章的文件名称，支持单次最多勾选20个附件。单个附件需满足pdf格式且小于100M。
- 文件列表：显示当前所有附件，支持复选框勾选/取消
- 已选数量提示：已选X条（最多20条）
- 操作按钮：取消、确定

#### 2.4.2 签署状态标记
点击确定后，关闭选择文件弹窗，在主弹窗的对应附件左上角显示"签署中"标签：
- 标签样式：橙色背景（#f5a623），白色文字，圆角
- 标签位置：附件卡片左上角，略微超出卡片边界（-top-2 left-2）
- 标签文字：签署中

### 2.5 底部提示
- 仅在**电子签章上传**方式下显示底部提示文字
- 提示内容：存在签章中/签章失败的文件时，请先点击"发起电子签后报名"完成电子签章。
