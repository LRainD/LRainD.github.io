# 资质检测列表

## 页面说明
资质检测功能的历史记录列表页，展示所有资质检测任务的执行记录和结果概况。

## 表格字段

| 字段 | 说明 |
|------|------|
| 状态 | 检测完成（绿色 #52C41A） |
| 资质检测编号 | 蓝色可点击链接，格式如 Q/20260415000013939 |
| 资质检测概况 | 显示"已通过 X 家，未通过 Y 家"，数字标蓝 |
| 检测发起时间 | 格式如 2026-04-15 19:24:52 |
| 权益使用 | 显示"消耗资质检测额度：X 家"，数字标蓝 |
| 创建时间 | 格式如 2026-04-15 19:24:43 |
| 发起人 | 显示用户名，如 jcftest1 (jcftest1) |
| 操作 | 查看（蓝色链接） |

## 操作说明

- **查看**：点击后跳转至检测详情页

## 数据示例

```typescript
{
  id: '1',
  status: 'completed', // 'completed'
  detectNo: 'Q/20260415000013939',
  overview: '已通过 0 家，未通过 1 家',
  startTime: '2026-04-15 19:24:52',
  quotaUsage: '消耗资质检测额度：1 家',
  createTime: '2026-04-15 19:24:43',
  initiator: 'jcftest1 (jcftest1)'
}
```
