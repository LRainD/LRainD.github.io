# 报名信息弹窗

- 文档编号：PRD-Registration-Info-Modal
- 负责人：[@liurundong]
- 版本：v1.4
- 状态：已完成

## 1. 摘要

报名信息弹窗，用于供应商在采购系统中提交报名信息。

## 2. 功能规格

### 2.1 联系方式
- 联系人（必填）
- 手机号（必填）
- 邮箱（必填）

### 2.2 报名附件

#### 2.2.1 附件是否使用电子签章（必填）
- 单选字段，选项：
  - **是**：使用电子签章上传，需进行电子签章后再提交
  - **否**：直接上传，传统附件上传方式
- 默认选中：**是**
- 选择后展开附件字段

#### 2.2.2 附件上传
- 附件上传（必填）

**选择"是"（电子签章）时：**
- 大小限制：100M
- 支持格式：.pdf
- 说明文字：电子签功能仅支持100M以内的pdf文件

**选择"否"（直接上传）时：**
- 大小限制：500M
- 支持格式：.jpeg、.jpg、.gif、.png、.doc、.docx、.xlsx 等
- 说明文字：大小限制：500M　支持格式：.jpeg、.jpg、.gif、.png、.doc、.doc_、.docx、.xl...（如需使用电子签功能，请上传100M以内的pdf文件）

### 2.3 操作按钮（联动规则）

按钮根据**附件是否使用电子签章**的选择进行联动：

| 附件是否使用电子签章 | 操作按钮 |
|---------------------|---------|
| 否 | 提交报名 |
| 是 | 发起电子签后报名 |

- 取消（始终显示）

### 2.4 电子签章流程

#### 2.4.1 提示弹窗
点击"发起电子签后报名"按钮后，弹出提示弹窗：
- 弹窗标题：提示（带橙色感叹号图标）
- 说明文字：以下附件将发起电子签章，签章完成后自动报名，最多支持20个文件发起签章
- 文件列表：显示当前所有附件，全部默认勾选，不可取消勾选
- 已选数量提示：已选X条（最多20条）
- 操作按钮：取消、确定

#### 2.4.2 签署状态标记
点击确定后，关闭提示弹窗，在主弹窗的对应附件左上角显示"签署中"标签：
- 标签样式：橙色背景（#f5a623），白色文字，圆角
- 标签位置：附件卡片左上角，略微超出卡片边界（-top-2 left-2）
- 标签文字：签署中

### 2.5 底部提示
- 仅在**选择"是"（电子签章）**方式下显示底部提示文字
- 提示内容：存在签章中/签章失败的文件时，请先点击"发起电子签后报名"完成电子签章。

### 2.6 提交报名反馈

#### 2.6.1 提交失败弹窗
当选择"否"（直接上传）点击"提交报名"时，如果附件列表中存在签署中/签章失败的文件，弹出提交失败弹窗：
- 弹窗标题：提交失败（带橙色感叹号图标）
- 提示内容：存在附件为签章失败/签章中状态，已为您暂存页面信息。签章中文件均签章成功时会自动为您完成报名（如果您已报名则会为您更新联系方式，但注意及时完成签章，否则报名截止时文件签章中/签章失败可能影响您的参与。如需撤销签章或签章附件错误您可删除文件后重新上传）
- 操作按钮：知道了

#### 2.6.2 报名成功提示
当选择"否"（直接上传）点击"提交报名"时，如果附件列表中不存在签署中/签章失败的文件，显示报名成功提示：
- 提示样式：顶部居中 Toast 提示
- 图标：绿色圆形勾选图标
- 提示文字：报名成功
- 关闭方式：点击右侧 X 按钮或 3 秒后自动消失
