# 招标工作台 - 报名与资审汇总页面规格文档

## 页面概述

招标工作台页面，展示项目招标流程及报名与资审相关信息。页面其他模块参照设计截图实现，本文档仅详细描述新增的「合作库分供商」模块规格。

---

## 模块：合作库分供商 ⭐ 新增模块

### 功能描述

展示合作库中的分供商列表，支持无限滚动分页加载。

### 位置

位于「报名与资审汇总」模块下方，作为独立模块展示。

### 界面元素

#### 1. 标题栏

| 元素 | 说明 |
|------|------|
| 左侧标题 | 「合作库分供商」 |
| 右侧按钮 | 「导出」按钮，点击后显示导出成功提示 |

#### 2. 分供商列表表格

**表格结构：**

| 列名 | 数据格式 | 说明 |
|------|----------|------|
| 分供商名称 | XX建材有限公司 | 随机生成，如：华夏建材、中建钢铁、东方水泥等 |
| 合作品类 | 物资 > 钢材 > 钢筋 | 三级层级结构，随机组合 |
| 联系方式 | 138****1234 | 手机号格式，中间4位脱敏处理 |

**表格样式：**

- 表头固定，背景色与主题一致（蓝色系）
- 内容区固定高度（约 400px），超出显示垂直滚动条
- 行高适中，保证可读性
- 斑马纹或悬停高亮效果

### 交互行为

#### 1. 无限滚动分页

```
初始加载 → 显示前20条数据
    ↓
用户滚动到底部
    ↓
显示 Loading 状态（固定2秒）
    ↓
加载下一页20条数据
    ↓
数据追加到列表末尾
    ↓
重复直到所有数据加载完毕
    ↓
底部显示「没有更多数据」提示
```

**Loading 状态：**
- 表格底部显示加载动画 + "加载中..." 文字
- 固定持续 2 秒后显示数据

**结束状态：**
- 所有数据加载完毕后，底部显示 "没有更多数据" 提示
- 不再触发加载

#### 2. 导出功能

- 点击「导出」按钮
- 显示导出成功提示（Message.success("导出成功")）
- 原型阶段无需真实导出文件

### 数据规格

#### 数据总量

模拟生成 100 条数据，分 5 页加载。

#### 数据生成规则

**分供商名称：**
- 格式：{前缀} + {行业} + 有限公司
- 前缀库：华夏、中建、东方、南方、北方、华东、华南、华西、华北、中原、长江、黄河、泰山、华山、恒信、永盛、宏达、鑫源、金鼎、银海
- 行业库：建材、钢铁、水泥、混凝土、砂石、管材、电缆、门窗、涂料、防水
- 示例：华夏建材有限公司、中建钢铁有限公司

**合作品类：**
- 格式：一级 > 二级 > 三级
- 一级：物资、设备、劳务、专业分包
- 二级（物资下）：钢材、水泥、混凝土、砂石、电缆、管材
- 三级（钢材下）：钢筋、型钢、钢板、钢管、钢丝
- 示例：物资 > 钢材 > 钢筋

**联系方式：**
- 格式：1xx****xxxx（中间4位脱敏）
- 前缀：130-139、150-159、180-189、198、199
- 示例：138****5678

### 状态说明

| 状态 | 说明 |
|------|------|
| 初始加载 | 页面加载时自动加载第一页数据 |
| 加载中 | 滚动到底部触发，显示 Loading 2秒 |
| 加载完成 | 新数据追加到列表 |
| 全部加载 | 100条数据加载完毕，显示结束提示 |
