# 计划详情页面规格文档

## 页面概述

计划详情页面是采购计划管理系统中的核心详情页，用于展示单个采购计划的详细信息和关联的采购条目列表。页面采用B端管理系统常见的左右布局，左侧为导航菜单，右侧为内容区域。

## 交互说明

### 侧边栏交互
1. 点击收起按钮，侧边栏宽度从200px变为64px
2. 收起状态下显示图标+缩写文字
3. 鼠标悬停菜单项显示背景色变化
4. 点击菜单项切换页面

### 标签页交互
1. 点击标签切换内容区域
2. 当前选中标签显示下划线和高亮颜色
3. 切换时有平滑过渡效果

### 表格交互
1. 表头固定，内容区域可横向滚动
2. 行悬停显示背景色变化
3. 点击选择框可多选行
4. 点击操作按钮显示更多操作菜单

### 查询区域交互
1. 点击下拉框展开选项列表
2. 点击查询按钮执行筛选
3. 点击重置清空查询条件
4. 点击展开显示更多查询条件

## 响应式说明

本页面主要针对桌面端设计，最小支持宽度1280px。在较小屏幕上：
- 表格区域支持横向滚动
- 查询条件自动换行
- 项目信息卡片信息自动换行
