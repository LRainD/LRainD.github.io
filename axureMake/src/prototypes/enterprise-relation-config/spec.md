# 企业关联检测配置页面 - 规格文档

## 概述

本文档记录企业关联检测配置页面的设计规格，重点描述企业名称模糊搜索功能的交互设计。

***

## 功能模块

### 1. 企业名称模糊搜索

#### 1.1 功能描述

用户在"企业名称"输入框中输入关键字后，系统提供智能模糊搜索建议，帮助用户快速选择企业并自动填充统一社会信用代码。

#### 1.2 交互流程

```
用户输入 → 防抖处理(500ms) → 触发搜索 → 展示下拉建议 → 用户选择 → 自动填充
```

#### 1.3 触发条件

| 条件         | 是否触发搜索 | 说明           |
| ---------- | ------ | ------------ |
| 输入为空       | ❌ 否    | 不显示下拉框       |
| 输入1个字符     | ❌ 否    | 字符数不足，不触发    |
| 输入2个及以上字符  | ✅ 是    | 满足最小长度要求     |
| 输入包含"公司"   | ❌ 否    | 关键词过滤，避免无效搜索 |
| 输入包含"有限公司" | ❌ 否    | 关键词过滤，避免无效搜索 |

#### 1.4 搜索结果展示

#### 1.5 视觉效果

**关键字高亮：**

- 匹配的关键字显示为红色 `#FF4D4F`
- 使用正则表达式匹配，不区分大小写

**悬浮效果：**

- 默认背景：白色
- 悬浮背景：浅蓝色 `#E6F4FF`
- 过渡：无过渡，即时响应

**选中效果：**

- 点击后下拉框关闭
- 企业名称填入输入框
- 统一社会信用代码自动填入对应字段

#### 1.6 防抖处理

- 延迟时间：500ms
- 触发时机：用户停止输入后500ms触发搜索

#### 1.7 边界处理

| 场景       | 处理方式            |
| -------- | --------------- |
| 点击外部区域   | 下拉框关闭           |
| 输入框失焦    | 保持下拉框显示（除非点击外部） |
| 无搜索结果    | 显示"暂无数据"        |
| 搜索中      | 显示"搜索中..."加载状态  |
