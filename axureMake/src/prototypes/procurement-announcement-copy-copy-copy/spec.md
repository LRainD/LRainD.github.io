# 编制公告页-设置报名附件检测 - 规格文档

## 📋 业务与功能

### 1.1 核心目标
在编制采购(资格预审)公告页面中，支持配置资格审查附件组成，并可选择是否启用AI辅助报名文件检测功能。启用后，可为每个附件设置需要AI检测的具体检测项。

### 1.2 功能清单

- **[资格审查附件组成配置]**：支持设定资格审查附件组成，包含文件项、文件项名称、是否必须上传、范本上传
- **[AI辅助报名文件检测开关]**：在「是否设定资格审查附件组成」为「是」时，显示「是否使用AI辅助报名文件检测」单选控件
- **[附件检测项设置]**：选择「是」后，附件表格联动增加「AI检测项设置」列，可为每行附件勾选需要检测的项
- **[检测项弹窗设置]**：点击「设置检测项」按钮，弹出小弹窗，按分类展示检测项，支持勾选/取消勾选

### 1.3 交互要点

- 「是否设定资格审查附件组成」选择「是」→ 显示附件表格和「是否使用AI辅助报名文件检测」单选
- 「是否使用AI辅助报名文件检测」选择「是」→ 附件表格增加「AI检测项设置」列
- 点击「设置检测项」/「已选N项」按钮 → 弹出检测项设置弹窗
- 弹窗内按分类展示检测项，勾选/取消勾选 → 实时更新该附件的检测项配置
- 点击「确定」或关闭按钮 → 弹窗关闭

## ⚙️ 数据与状态

### 2.1 AI辅助报名文件检测状态

| 状态 | 说明 |
|------|------|
| `isAIFileDetectSet` | 是否启用AI辅助报名文件检测，默认 `false` |
| `attachmentCheckSettings` | 每个附件对应的检测项配置，`Record<number, string[]>` |
| `checkModalAttachmentId` | 当前打开检测项设置弹窗的附件ID |

### 2.2 检测项分类

检测项按分类组织，便于扩展：

| 分类 | 检测项 |
|------|--------|
| 签章类 | 是否盖章、法人签字、骑缝章 |
| 时效类 | 有效期校验、年检状态 |
| 资质类 | 颁发机构核验、资质等级匹配 |

### 2.3 联动规则

```
是否设定资格审查附件组成 = 是
    ↓
显示附件表格 + 显示「是否使用AI辅助报名文件检测」
    ↓
是否使用AI辅助报名文件检测 = 是
    ↓
附件表格增加「AI检测项设置」列
```

## 🎨 UI 说明

### 3.1 「是否使用AI辅助报名文件检测」单选

- 位置：资格审查附件组成区域，附件表格上方
- 样式：与现有单选控件保持一致（Radio.Group）
- 选项：是 / 否

### 3.2 「AI检测项设置」列

- 位置：附件表格中，「范本」列之后，「操作」列之前
- 默认显示：「设置检测项」按钮（带 Settings 图标）
- 已配置后显示：「已选N项」按钮

### 3.3 检测项设置弹窗

- 标题：检测项设置 - {附件名称}
- 布局：按分类分组展示，每类一个区块
- 分类标题：带左侧色条的小标题
- 检测项：每分类下两列网格排列的复选卡片
- 底部：左侧显示「已选 N 项」，右侧「确定」按钮
