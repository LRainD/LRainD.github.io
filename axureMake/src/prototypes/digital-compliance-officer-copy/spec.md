## 1. 摘要

### 1.1 一句话说明

为运营人员提供集采AI能力的统一配置入口，支持对不同组织灵活开启"风控预警"、"智能问答"等AI功能模块，并为各模块提供精细化配置能力，为后续AI能力扩展预留空间。

## 2. 背景与问题

### 2.1 背景

- 业务背景：在招投标系统中，采购商在招标过程中（如发布公告、编制招标文件、评标等）可能存在不合规风险。为了降低合规风险，引入"风控预警"等AI能力进行自动化校验。随着AI技术的发展，未来还将引入"智能问答"等更多AI功能模块。
- 用户背景：运营人员需要能够灵活地为不同的采购组织（企业/机构）开启AI功能，并根据不同组织的业务特点配置不同的功能模块和规则。
- 技术/合规背景：基于AI能力中台架构，为不同业务场景提供可配置、可扩展的智能化服务。

### 2.2 现状与痛点

| 角色/场景     | 当前流程                | 痛点                 | 影响              |
| --------- | ------------------- | ------------------ | --------------- |
| 运营人员/配置场景 | 缺乏统一的AI能力配置入口，各功能独立配置 | 无法统一管理AI功能模块，配置效率低，扩展性差 | 响应业务需求慢，增加运维成本  |
| 采购商/业务流程  | 依赖人工审核或事后审计         | 无法在业务节点实时发现不合规信息   | 产生法律风险，可能导致招标无效 |

## 3. 用户与场景

### 3.1 目标用户（Personas）

| Persona | 关键特征           | 主要目标                        | 主要阻碍                  | 触达渠道 |
| ------- | -------------- | --------------------------- | --------------------- | ---- |
| 运营人员    | 负责系统的组织管理与功能配置 | 能够灵活地为不同采购商开启AI功能模块，并进行个性化设置 | 配置项不全、无法批量控制下级、权限管控不严 | 运营后台 |

### 3.2 核心使用场景（Top Scenarios）

1. **场景A：新组织AI功能开启**：运营人员在后台搜索到某个采购组织 -> 点击新增配置 -> 选择该组织 -> 设置范围为"本下级" -> 勾选功能模块（如：风控预警、智能问答） -> 配置AI别名和头像 -> 针对风控预警模块配置业务系统和关联人员 -> 保存生效。
2. **场景B：配置范围调整**：某组织原本只开启了"本级"，现在需要调整为"本下级"以覆盖子公司。
3. **场景C：功能模块扩展**：某组织原本只开启了"风控预警"，现在需要新增"智能问答"模块。
4. **场景D：AI形象定制**：为不同组织配置个性化的AI别名和头像，提升品牌识别度。

### 3.3 用户故事（User Stories）

- 作为运营人员，我想要能够选择配置的范围（本级/本下级），以便更高效地管理组织树下的AI功能。
- 作为运营人员，我想要多选功能模块（风控预警/智能问答），以便灵活控制不同AI能力的开启。
- 作为运营人员，我想要为AI助手配置统一的别名和头像，以便在业务系统中展示更具识别度的形象。
- 作为运营人员，我想要针对"风控预警"模块配置业务系统和关联人员，以便精细化控制风控功能的覆盖范围。
- 作为运营人员，我需要特定的权限点才能访问配置页面，以便确保系统配置的安全性。
- 作为运营人员，我希望配置架构能够支持未来扩展更多AI功能模块，以便快速响应业务需求。

### 3.4 用户旅程/流程（User Flow）

1. **进入管理后台**：运营人员登录 -> 导航至"配置管理 -> 集采AI配置"模块（需权限校验）。
2. **列表管理**：在配置列表页查看已有配置 -> 支持搜索组织 -> 点击"新增配置"。
3. **表单填写**：
   - **基本配置**：
     - **组织选择**：从组织树中选择一个目标组织。
     - **范围控制**：选择"本级"或"本下级"。
     - **功能模块选择**：多选框勾选（目前支持：风控预警、智能问答）。
     - **AI形象配置**：配置AI别名、上传AI头像。
   - **定制配置**（当勾选"风控预警"时显示）：
     - **风控预警开启业务系统**：多选框勾选（目前支持：招投标、合同）。
     - **关联人员配置**：选择目前用户中心维护的采购商类人员，支持多选。
4. **提交生效**：点击保存 -> 系统校验（如组织是否已配置、功能模块是否选择） -> 提示成功 -> 返回列表页。

- 校验失败：

  - 功能模块勾选"风控预警"但未勾选业务系统时提示"请至少选择一个业务系统"。
  - 功能模块勾选"风控预警"但未选择关联人员时提示"请选择关联的风控人员"。
